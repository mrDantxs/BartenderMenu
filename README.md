#  BartenderMenu (Android)

Aplicativo Android desenvolvido no Android Studio para gerenciamento, cadastro e degustação de drinks.

## Status atual do projeto
O aplicativo já é funcional e possui as seguintes telas implementadas:

- Tela inicial (Main)
- Introdução em slides (personalizada)
- Lista de Drinks
- Cadastro de novos Drinks
- Tela de Degustação
- Tela de Resultados
- Geração de PDF com informações dos drinks

##  Tecnologias utilizadas
- Android Studio
- Java
- XML para layouts
- Armazenamento de arquivos locais (imagens dos drinks)
- Geração de PDF personalizada

##  Estrutura do projeto
O código está organizado em pacotes por responsabilidade:

- `ui/` → telas do aplicativo
- `model/` → classes de modelo (Drink)
- `database/` → gerenciamento do banco local
- `utils/` → utilitários (FileUtils e PdfGenerator)
